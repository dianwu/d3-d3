# D3 D3

Diablo3 with D3js

## Getting Started
Download the [production version][min] or the [development version][max].

[min]: https://raw.github.com/dianwu/d3-d3/master/dist/d3-d3.min.js
[max]: https://raw.github.com/dianwu/d3-d3/master/dist/d3-d3.js

io page:http://dianwu.github.io/d3-d3/

demo page: http://dianwu.github.io/d3-d3/d3.html

In your web page:

```html
<script src="jquery.js"></script>
<script src="dist/d3-d3.min.js"></script>
<script>
jQuery(function($) {
  $.awesome(); // "awesome"
});
</script>
```
